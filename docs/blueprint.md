# **App Name**: Tijuana Marketplace

## Core Features:

- Product Listing: Allow users to list products with photos, title, description, price, category, condition, and location.
- User Registration and Login: Enable users to create accounts and log in securely to access the app's features. Registration requires name, email, password, and optional profile photo.
- Product View: Display product details with a photo carousel, detailed description, price, category, condition, location, and seller profile.
- User Profiles: Allow users to view and edit their profile information, including photo, name, location, and contact details.
- Search and Filtering: Implement a search functionality with filters for category, price range, condition, and sorting options.
- Chat Functionality: Enable direct messaging between buyers and sellers, including text and image options, with push notifications.
- Favorites: Allow users to add products to a list of favorites.

## Style Guidelines:

- Primary color: A vibrant orange (#FF7849) to evoke feelings of excitement and energy in a diverse marketplace. It is neither overly corporate nor stereotypically 'tech'.
- Background color: A light, desaturated orange (#F8EFE9) that creates a warm and inviting backdrop.
- Accent color: A deep red (#B83328) is employed as an accent, creating contrast that highlights crucial calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a balance of modernity and warmth for readability and user engagement.
- Use clean, simple icons to represent product categories and features.
- Implement a clear and intuitive layout for easy navigation and product discovery.
- Subtle animations to indicate actions like adding to favorites or sending a message.
